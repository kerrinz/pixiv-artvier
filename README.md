## 技术栈

基于[`Flutter`](https://flutter.dev)构建。使用[`riverpod`](https://github.com/rrousselGit/riverpod)作为状态管理，[`intl`](https://pub.flutter-io.cn/packages/intl)适配国际化多语言。

## 简介

一个 pixiv 第三方 app，同时支持Android、IOS系统，仅用于学习交流使用。

## 安装方式
目前还在开发中，打算完善度较高后再发布。
